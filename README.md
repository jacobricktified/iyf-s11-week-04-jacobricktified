## Project Description

This project introduces the fundamentals of JavaScript programming. It covers variables, data types, operators, functions, control flow, arrays, objects, and data manipulation techniques. The project includes two mini-projects: an interactive calculator and a student grade tracker, along with several coding challenges designed to strengthen problem-solving skills.

## Technologies Used

* HTML5
* JavaScript (ES6+)
* Visual Studio Code
* Git
* GitHub

## Features

* JavaScript variable and data type practice
* Arithmetic, comparison, and logical operators
* Function declarations, expressions, and arrow functions
* Conditional statements and loops
* Interactive calculator supporting multiple operators
* Array manipulation using map, filter, find, and reduce
* Object creation and manipulation
* Student Grade Tracker system
* Daily coding challenges (FizzBuzz, Reverse String, Largest Number, Remove Duplicates, Palindrome Checker)

## Lessons Learned

During this project, I learned:

* The differences between `let`, `const`, and variable scope.
* How JavaScript handles different data types.
* How to create and use functions effectively.
* How conditional statements and loops control program flow.
* How arrays and objects store and organize data.
* How to use array methods such as `map()`, `filter()`, `find()`, and `reduce()`.
* How to build reusable and modular JavaScript programs.
* How to solve common coding challenges using JavaScript.

## Challenges Faced

### Challenge 1: Understanding Array Methods

Initially, distinguishing between `map()`, `filter()`, and `reduce()` was difficult. This was solved through repeated practice and implementing real-world examples.

### Challenge 2: Object Manipulation

Working with nested objects and iterating through object properties required extra practice. Using `Object.keys()`, `Object.values()`, and `Object.entries()` improved understanding.

### Challenge 3: Building the Student Grade Tracker

Calculating averages, identifying top performers, and generating report cards required combining arrays, objects, loops, and functions. Breaking the project into smaller methods made implementation easier.
